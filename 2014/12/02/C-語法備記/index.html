C# 語法備記

很久以前就想碰C#了，剛好現在工作上需要用到，在感受他強大的威力之前，先來了解一下基本語法吧!

Array

建立一個陣列空間

資料型態[] 陣列名稱 = new 資料型態[陣列大小];
建立一個長度為5的整數陣列

int[] IntArray = new int[5];

建立陣列並給初始值

int[] IntArray = new int[]{1,2,3,4,5};

Foreach

取出陣列中每個內容

foreach ( var i in x){
    // i = 陣列內容
    // x = 陣列
}

List

建立一個List
List<資料型態> list1 = new List<資料型態>();

建立一個Image List

List<Image> ListName = new List<Image>();

List使用

ListName.Add(object);    //將object加入List中
ListName.Count ;         //取得目前List內容數量

Class

要自己建立Class 直接寫在主要Class 下面就可以用了
不可以寫在上面

namespace text1
{
    public partial class UserControl1 : UserControl
    {
        public UserControl1()
        {
            InitializeComponent();

        }

    }
    class MyClass {

        //your code
    }
}
